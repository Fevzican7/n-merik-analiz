Fibonacci Dizisi

Antik çağlardan modern bilgisayar bilimlerine uzanan bu gizemli sayı dizisinin ardındaki algoritmik gücü ve matematiksel kusursuzluğu inceleyin.

Sınıf Algoritması Uygulamalı Deneyler

Leonardo of Pisa (Fibonacci)

Erken Dönem ve Hint-Arap Rakamları

Tam adıyla Leonardo Bonacci (1170 - 1250), İtalya'nın Pisa şehrinde doğmuştur. Babası Guglielmo, Kuzey Afrika'daki Bugia limanında bir tüccardı. Babasının yanına giderek orada eğitim alan genç Leonardo, Arap matematikçilerden günümüzde kullandığımız 10 tabanlı (0-9) Hint-Arap rakam sistemini öğrendi. Bu sistem, o dönem Avrupa'da hesaplama yapmayı imkansız kılan hantal Roma rakamlarına göre devrim niteliğindeydi.

Liber Abaci ve Modern Matematiğin Doğuşu

1202 yılında Pisa'ya dönen Fibonacci, bu sistemi detaylandırdığı başyapıtı Liber Abaci'yi yayımladı. Sıfır (0) rakamının basamak değeri olarak kullanılması, Avrupa'da modern matematik ve bilgisayar algoritmalarının temellerini attı.

Ünlü dizi ise bu kitapta yer alan "Tavşan Üreme Problemi" ile ortaya atıldı. "Her ay yeni bir çift doğuran tavşanların 1 yıl sonundaki sayısı ne olur?" sorusu, Altın Oran'ın formülünü matematik literatürüne kazandırdı.

İlahi Oran: Altın Oran (Phi)

Altın Oran Nedir?

Altın Oran (Phi, 1.618...), matematikte ve sanatta, bir bütünün parçaları arasındaki en estetik ve en dengeli oran olarak kabul edilir. İki miktar arasındaki oran, bu iki miktarın toplamının büyük olanına oranına eşitse, bu oran "Altın Oran"dır.

Fibonacci dizisindeki ardışık iki sayının oranı, sayılar büyüdükçe kusursuz bir şekilde bu gizemli sabite yakınsar: 34/21 = 1.619, 55/34 = 1.617, 89/55 = 1.618...

Tarihte Nasıl Bulundu?

Tarihte ilk kez Antik Yunan'da ünlü matematikçi Öklid (M.Ö. 300) tarafından "Ekstrem ve Ortalama Oran" olarak tanımlanmıştır. Ancak mimarideki altın dikdörtgen oranları sebebiyle antik tapınakların (örneğin Partenon) inşasında da kullanıldığı gözlemlenmektedir.

Rönesans döneminde Luca Pacioli'nin yazdığı ve Leonardo da Vinci'nin resimlediği "De Divina Proportione" (İlahi Oran) adlı kitapla tüm sanat ve mimari dünyasına yayılmıştır. Günümüzde ise modern yazılım arayüzü tasarımlarının vazgeçilmezidir.

Fibonacci Search (Arama Algoritması)

İkili Arama (Binary Search) bölme işlemi kullanırken, Fibonacci Arama sadece toplama ve çıkarma işlemleri yaparak donanım seviyesinde çok daha verimli çalışır.

Soru: Sıralı bir dizide x sayısını arayan Fibonacci Search algoritmasını yazınız.
  1. BAŞLA
  2. EĞER Dizi Boşsa VEYA Hatalıysa { YAZ "Hata", BİTİR }
  3. fbM2 = 0, fbM1 = 1, fbM = 1
  4. Döngü (fbM < n) { fbM2=fbM1; fbM1=fbM; fbM=fbM2+fbM1; } // Sınırı bul
  5. offset = -1
  6. Döngü (fbM > 1)
    {
      i = min(offset + fbM2, n - 1);
      EĞER (arr[i] < x) { fbM=fbM1; fbM1=fbM2; fbM2=fbM-fbM1; offset=i; } // Sağ yarı
      EĞER (arr[i] > x) { fbM=fbM2; fbM1=fbM1-fbM2; fbM2=fbM-fbM1; } // Sol yarı
      DEĞİLSE { YAZ "Bulundu"; BİTİR; }
    }
  7. EĞER (fbM1 == 1 VE arr[offset+1] == x) { YAZ "Bulundu"; BİTİR; }
  8. YAZ "Bulunamadı", BİTİR

FibonacciSearch.java 
public int fibonacciSearch(int[] arr, int x) {
    if (arr == null || arr.length == 0) return -1;
    int n = arr.length;
    
    int fb2 = 0; // (m-2). Fibonacci
    int fb1 = 1; // (m-1). Fibonacci
    int fbM = fb2 + fb1; // m. Fibonacci

    while (fbM < n) {
        fb2 = fb1;
        fb1 = fbM;
        fbM = fb2 + fb1;
    }

    int offset = -1;
    while (fbM > 1) {
        int i = Math.min(offset + fb2, n - 1);
        if (arr[i] < x) {
            fbM = fb1; fb1 = fb2; fb2 = fbM - fb1;
            offset = i;
        } else if (arr[i] > x) {
            fbM = fb2; fb1 = fb1 - fb2; fb2 = fbM - fb1;
        } else {
            return i; // Bulundu
        }
    }
    
    if (fb1 == 1 && offset + 1 < n && arr[offset + 1] == x)
        return offset + 1;
        
    return -1; // Bulunamadı
}

Pascal Üçgeni ve Fibonacci Bağı

Blaise Pascal ve Tarihçesi

Blaise Pascal (1623-1662), Fransız matematikçi ve fizikçidir. Olasılık kuramının temellerini atan bilim insanlarından biridir.

Bu üçgenin özellikleri Pascal'dan yüzyıllar önce Hint, İranlı (Ömer Hayyam) ve Çinli matematikçiler tarafından biliniyordu. Ancak Pascal'ın bu konudaki "Traité du triangle arithmétique" adlı eseri o kadar kapsamlıydı ki, Batı dünyasında bu yapı onun adıyla anıldı.

Matematiğin En Büyük Sürprizi

Pascal üçgenindeki her sayı, üstündeki iki sayının toplamıdır (Kombinasyon mantığı). Ancak bu üçgenin içine gizlenmiş muazzam bir sır vardır: Fibonacci Dizisi!

Eğer Pascal üçgenindeki sığ köşegenleri (çapraz satırları) toplarsanız, kusursuz bir şekilde Fibonacci sayılarını elde edersiniz (1, 1, 2, 3, 5, 8, 13...). Evrenin farklı gibi görünen iki matematiksel konsepti, aynı potada birleşmektedir.

Soru: Pascal üçgeninin ilk N satırını çizen algoritmayı yazınız.
  1. BAŞLA
  2. OKU N // Satır sayısını al
  3. EĞER N < 1 VEYA N > 15 İSE
        YAZ "Hata: 1-15 arası girin" VE BİTİR
  4. Döngü (i = 0; i < N; i++)
    {
        val = 1;
        Döngü (boşluk = 0; boşluk < N - i; boşluk++) { YAZ " "; }
        Döngü (j = 0; j <= i; j++)
        {
            YAZ val;
            val = val * (i - j) / (j + 1);
        }
        YAZ YeniSatır;
    }
  5. BİTİR

PascalTriangle.java 
import java.util.Scanner;

public class PascalTriangle {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        
        // Hata Kontrolü
        if(n < 1 || n > 15) {
            System.out.println("Hata: Lütfen 1-15 arası girin");
            return;
        }

        // O(1) Extra Space, O(N^2) Time Complexity
        for (int i = 0; i < n; i++) {
            // Görsel hizalama (Boşluklar)
            for (int space = 0; space < n - i; space++) {
                System.out.print("  ");
            }

            int val = 1;
            for (int j = 0; j <= i; j++) {
                System.out.printf("%4d", val);
                // Kombinasyon: C(n, k+1) = C(n, k) * (n-k) / (k+1)
                val = val * (i - j) / (j + 1);
            }
            System.out.println();
        }
    }
}

Büyük Sayı Jeneratörü

Java'nın BigInteger mimarisi ile yazılmış ve alanı O(1)'e düşüren, taşma hatasını (overflow) çözen devasa sayı üretim algoritması.

Soru: N=1000 için taşma yapmayan algoritmayı yazınız.
  1. BAŞLA
  2. OKU N
  3. EĞER N < 1 VEYA N > 1000 İSE
        YAZ "Hata: 1-1000 arası değer giriniz" VE BİTİR
  4. BigInt a = 0, b = 1; // O(1) Alan Optimizasyonu
  5. Döngü (i = 1; i <= N; i++)
    {
        YAZ a;
        BigInt next = a + b;
        a = b;
        b = next;
    }
  6. BİTİR
FibonacciGenerator.java 
import java.util.Scanner;
import java.math.BigInteger;

public class FibonacciGenerator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int n = scanner.nextInt();

        // Sınır Kontrolü
        if (n <= 0 || n > 1000) {
            System.out.println("Hata: Lütfen 1-1000 arası girin");
            return;
        }
        
        BigInteger a = BigInteger.ZERO;
        BigInteger b = BigInteger.ONE;

        for (int i = 1; i <= n; i++) {
            System.out.println(i + ": " + a);
            BigInteger next = a.add(b);
            a = b;
            b = next;
        }
    }
}

Tarihteki ilk Fibonacci problemi

Aylara göre üreyen tavşan çifti sayısını simüle edin.

-

Algoritma (Dinamik Programlama - Tabulation): Eğer özyineli (recursive) yöntem kullansaydık hesaplamalar tekrarlanacağı için zaman karmaşıklığı O(2^n) olurdu. Bunun yerine hesaplanan değerler bir dp dizisinde saklanır. Böylece hesaplama O(n) sürede tamamlanır.

RabbitPopulation.java
public long rabbitPairs(int months) {
    if (months <= 1) return 1;
    long[] dp = new long[months + 1];
    dp[0] = 1; dp[1] = 1;
    for (int i=2; i<=months; i++) 
        dp[i] = dp[i-1] + dp[i-2];
    return dp[months];
}

05

Kriptografi (Şifreleme)

Metni Fibonacci dizisiyle şifrele / çöz
▼

Metnin her harfini, bulunduğu sıradaki Fibonacci sayısı kadar alfabede kaydırarak şifreleyin ve aynı mantıkla çözün.

-
-

Algoritma (Dinamik Sezar Şifrelemesi): Standart şifrelemede her harf sabit bir miktar kaydırılırken, burada kaydırma miktarı (shift key) Fibonacci serisi ile belirlenir. Metnin i. karakteri, alfabede F(i) modulo 26 kadar ilerletilir. Şifre çözülürken aynı işlem tersine (eksi yönlü) yapılır.

FiboCipher.java
public class FibonacciCipher {
    // Şifreleme
    public static String encrypt(String text, int[] fib) {
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < text.length(); i++) {
            char c = text.charAt(i);
            if (c >= 'A' && c <= 'Z') {
                int shift = fib[i] % 26;
                sb.append((char)('A' + (c - 'A' + shift) % 26));
            } else sb.append(c);
        }
        return sb.toString();
    }
    
    // Deşifreleme
    public static String decrypt(String text, int[] fib) {
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < text.length(); i++) {
            char c = text.charAt(i);
            if (c >= 'A' && c <= 'Z') {
                int shift = fib[i] % 26;
                // Eksi değerleri önlemek için +26 eklenir
                sb.append((char)('A' + (c - 'A' - shift + 26) % 26));
            } else sb.append(c);
        }
        return sb.toString();
    }
}

Merdiven Tırmanma Kombinasyonları

Adım varyasyonlarını hesapla
▼

N basamaklı merdiveni 1 veya 2 adım atarak kaç farklı şekilde çıkarsınız?

-

Algoritma (Kombinatoryal Fibonacci): N. basamağa ancak (N-1). veya (N-2). basamaktan adım atılabilir. Bu yüzden N basamağa ulaşım yollarının sayısı, önceki iki basamağa ulaşım yollarının toplamıdır. Bu, direkt olarak Fibonacci serisinin tanımıdır.

ClimbingStairs.java
public int climbStairs(int n) {
    if (n <= 2) return n;
    int a=1, b=2;
    for (int i=3; i<=n; i++) {
        int temp = a+b; a=b; b=temp;
    } return b;
}

const ctx = new AudioContext();

function fiboToFreq(fiboNum, scale) {
  const baseFreq = 220; // A3 = 220 Hz

  const gain = ctx.createGain();

Sıkça Sorulan Sorular

1. Fibonacci dizisi tam olarak nedir?

İlk iki terimi 0 ve 1 olan, sonraki her terimin kendinden önceki iki terimin toplamıyla elde edildiği sayı dizisidir.

2. Java'da 1000. terim için neden BigInteger şarttır?

Standart "long" değişkeni maksimum 93. Fibonacci terimini alabilir. 1000. terim ise 209 basamaklıdır, özel kütüphane gerektirir.

3. Nümerik Analiz derslerinde neden öğretilir?

Farklı algoritma tasarımlarının (Yinelemeli vs. Dinamik Döngüsel) işlemciyi ve belleği nasıl etkilediğini gösteren laboratuvar örneğidir.

4. Altın Oran ile Fibonacci dizisi nasıl ilişkilidir?

Ardışık iki Fibonacci sayısının oranı (F(n)/F(n-1)), n büyüdükçe Altın Oran'a (yaklaşık 1.6180339887) yakınsar. Bu yakınsama, dizinin en büyüleyici özelliklerinden biridir.

5. Pisano Periyodu ne işe yarar?

Büyük Fibonacci sayılarını modüler aritmetikle hesaplarken periyodan faydalanılır. Kriptografi ve bilgisayar bilimlerinde önemli uygulamaları vardır.

6. Fibonacci Search'ün avantajı nedir?

İkili arama bölme işlemi kullanırken, Fibonacci Search sadece toplama ve çıkarma kullanır. Donanım seviyesinde çok daha az enerji tüketir ve hızlı çalışır.

7. Doğada bu dizi nerelerde görülür?

Ayçiçeği çekirdeklerinin diziliminde, çam kozalaklarında, ananas kabuklarında, DNA sarmalında ve sarmal galaksilerin yapısında kusursuzca görülür.
